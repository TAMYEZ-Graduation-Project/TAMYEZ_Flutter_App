# Career Compass

An AI-powered career guidance mobile application built with Flutter that helps users discover suitable career paths through intelligent assessments, personalized recommendations, interactive learning roadmaps, and dynamic quizzes.

The application guides users from career exploration to structured learning by combining AI-generated assessments, personalized career recommendations, roadmap-based learning, and progress tracking in a seamless mobile experience.

---

## ✨ Features

### Authentication & Account Management
- Email & Password Authentication
- Google Sign-In
- Forgot Password Flow
- Email Verification
- Secure Session Management

### AI-Powered Career Assessment
- Dynamic assessment questions
- Interactive assessment experience
- Personalized analysis based on user responses

### Career Recommendations
- Receive top career recommendations
- AI-assisted career matching
- Guided career selection process

### Personalized Learning Roadmaps
- Structured learning paths
- Step-by-step progression
- Locked and unlocked learning stages

### Dynamic Quizzes
- Quiz-based progression system
- Knowledge validation after each stage
- Roadmap unlocking based on performance

### Progress Tracking
- Track completed roadmap steps
- Save learning progress locally
- Continue learning from where you stopped

### Notifications
- Push notifications using Firebase Cloud Messaging
- Local notifications using Awesome Notifications

### User Profile
- Manage account information
- View learning progress
- Track completed activities

# 🏗️ Architecture

This project follows **Clean Architecture** principles combined with an **MVI (Model-View-Intent)** approach to create a scalable, maintainable, and testable codebase.

### Architectural Concepts

- Clean Architecture
- MVI (Model - View - Intent)
- SOLID Principles
- Repository Pattern
- Dependency Injection
- Separation of Concerns
- Unidirectional Data Flow
- Feature-Based Modularization

### Layers

#### Presentation Layer

Responsible for rendering UI, handling user interactions, and state management.

Contains:

- Screens
- Widgets
- ViewModels (Cubit)
- Intents
- UI State

#### Domain Layer

Contains the business logic of the application.

Contains:

- Entities
- Repository Contracts
- Use Cases

#### Data Layer

Responsible for data retrieval and persistence.

Contains:

- Data Sources
- Models
- Mappers
- Repository Implementations

---

# 🧠 State Management

State management is implemented using **Flutter Bloc (Cubit)** while following an **Intent-Driven MVI Architecture**.

### MVI Flow

```text
View
 ↓
Intent
 ↓
ViewModel (Cubit)
 ↓
State
 ↓
View
```

### Benefits

- Predictable state updates
- Clear data flow
- Better separation of concerns
- Easier debugging
- Improved testability
- Scalable feature development

### State Handling

Each feature maintains:

- Loading States
- Success States
- Error States
- Side Effects

UI side effects such as navigation and messages are separated from UI state to keep the architecture clean and maintainable.

# 🚀 Getting Started

## Prerequisites

- Flutter SDK
- Dart SDK
- Android Studio or VS Code
- Firebase Project Configuration
- Backend API

## Clone Repository

```bash
git clone https://github.com/your-username/career-compass.git
```

## Install Dependencies

```bash
flutter pub get
```

## Generate Code

```bash
dart run build_runner build --delete-conflicting-outputs
```

## Run Application

```bash
flutter run
```
